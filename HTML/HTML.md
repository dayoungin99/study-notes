## Basic Elements
1. Tag
  - Form: <TagName>
2. Attributions
  - Form: <TagName AttrName="AttrValue">
3. Symtax
  a) Content exists:
    - ex) <title>My First Web Page!</title>
      -> <title>: Open tag
      -> My First Web Page!: Content
      -> </title>: Close tag
  b) Content NOT exists:
    - empty tag: <br>
4. Comment
  - Form: <!-- Comment Content -->
